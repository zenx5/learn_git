Aqui las actividades: 
1) Borrar el archivo "otrascosa.txt" 
    - Tan sencillo como seleccionarlo, click derecho y remove
    - tambien podrias hacerlo desde la terminar 
      con este comando: "rm -r otrascosa.txt" pero no es necesario
2) Luego de eso has un "git status", notaras que hay cambios sin ser confirmados.
   Deberas hacer uso del Flujo, recuerda:
   - git add otrascosa.txt
   - git commit -m "algun mensaje aqui"
   - git pull
   - git push
3) Ok ya has borrado archivos y confirmados cambios, ahora vamos a modificar
   el archivo README.md, en la seccion de "Others Usefull Commands" u "Otros Camandos Utiles"
   debemos agregar el comando "git checkout" que usaste para llegar a esta rama,
   deberias agregar al final de esta seccion algo asi: "git checkout [BRANCH]"
   Luego de esto guardalo y aplica nuevamente el flujo.
4) Luego de esto vamos al archivo "biografia.txt", debes corregir algunos errores que le agregue
   Cuando las correcciones esten hechas, guardas y vuelves a aplicar el flujo para confirmar esos cambios
